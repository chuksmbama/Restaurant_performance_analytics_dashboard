# Restaurant_performance_analytics_dashboard
## Overview
The objective of this analysis is to evaluate overall restaurant performance by examining sales trends, customer growth, product performance and purchasing behaviour. The goal is to provide clear insights that support strategic decicison making, improve revenue and identify areas of operational opportunity.
